# Moregooder [backend]

## Features
- Clean Architecture / Hexagonal Architecture (Ports and Adapters)
- Firestore DB
- TypeScript
- Integrtion and unit testing (TDD)
- Documentation with Swagger
- CI with Github Actions

## Deploy

```
gcloud config set project moregooder
```
